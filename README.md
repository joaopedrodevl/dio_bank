
# DIO Bank

Projeto do curso FullStack da DIO para criar uma app simples de banco.

## Stack utilizada

**Back-end:** Node, TypeScript

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/joaopedrodevl/dio_bank.git
```

Entre no diretório do projeto

```bash
  cd dio_bank
```

Instale as dependências

```bash
  npm i
```

Rode o programa

```bash
  npm run dev
```
